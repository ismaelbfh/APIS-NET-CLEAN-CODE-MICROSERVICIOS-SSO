# ARQUITECTURA MICROSERVICIOS .NET PROPIA ORIENTADA AL DOMINIO DDD - NEGOCIO
Esto es una arquitectura en .NET CORE con mucho mimo y mucha dedicación, se puede integrar en cualquier negocio que implemente microservicios, 
metodo de autenticacion SSO a traves de una unica API que autentica a las demás con ese token, tiene refrescos de token, logica de negocio,
patrones de diseño como el Repository, buenas practicas con Integration tests, CLEAN CODE, SOLID.
Son 3 APIS cada una interactua con su propio csproj de negocio.
Tambien incluyo un archivo que es azure-pipelines.yaml para que se vea como hice para desplegar en entornos productivos la aplicacion (Solo genero ARTIFACT-ZIP y luego en RELEASE LO SUBO AL SERVIDOR IIS MEDIANTE UNAS REGLAS.
